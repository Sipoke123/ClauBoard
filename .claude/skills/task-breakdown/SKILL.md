# Purpose

Use this skill whenever a large feature or project needs execution planning.

# Required outputs

For any large task, produce:
- milestone list
- ordered implementation tasks
- dependencies
- acceptance criteria
- risks
- recommended first slice

# Rules

- Break work into tasks that can be completed in one focused session.
- Prefer vertical slices over horizontal endless scaffolding.
- Each task must have a concrete artifact or code outcome.
- Mark blockers explicitly.

# For this project

Prefer milestones such as:
1. define product and architecture
2. scaffold repo
3. define shared event schema
4. create backend event gateway
5. create MVP operator UI
6. connect a mock Claude agent adapter
7. replace mock adapter with real Claude Code integration
